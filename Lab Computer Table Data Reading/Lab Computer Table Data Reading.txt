# 表格数据读取程序使用说明

1、安装数据读取软件：进入网站https://github.com/UB-Mannheim/tesseract/wiki，点击‘tesseract-ocr-w64-setup-5.3.3.20231005.exe’安装程序

2、使用vscode在读取结果欲保存的路径下打开程序。

3、配置程序中所需的各个函数包。

4、将‘表格数据读取.py’中的第10行中的文件路径修改为tesseract.exe的实际路径。

5、运行程序，按照程序提示，分别用鼠标左、右键标定待提取表格的左上角、右下角。

6、关掉程序弹出的提取预览图片。

7、结果存储至预设路径，打开查看。
